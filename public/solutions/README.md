# Section 3 — Solution Images

Put the 4 slide images for "Existing Industrial Inspection Solutions" in this folder.

## Rules
- Use these **exact filenames**:
  - `xray-inspection.jpg`        → Internal & Subsurface Defect Detection (blue)
  - `machine-vision.jpg`         → External Surface & Cosmetic Defect Detection (purple)
  - `cmm-laser-inspection.jpg`   → Dimensional, Geometric & Assembly Inspection (green)
  - `acoustic-testing.jpg`       → Acoustic & Dynamic Integrity Testing (orange)
- **JPG** format is preferred.
- Recommended size: **900 × 600 px** (or any clean 16:10 landscape). Images are cropped with `object-cover`.
- If an image is missing, a clean color-matched placeholder icon shows automatically — so the section always renders.

## Changing format (optional)
To use .png/.webp, edit the `image:` paths in `src/data/presentationData.js`
(the `EXISTING_SOLUTIONS` array) and replace `.jpg` with your extension.
